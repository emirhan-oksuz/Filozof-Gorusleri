# Filozof Görüşleri

Bu proje, çeşitli filozofların kısa düşüncelerini kart yapısı içinde gösteren basit bir web sitesidir.

## Özellikler

- Kart tabanlı basit kullanıcı arayüzü
- Filozof görsellerinin çoğu JSON dosyasındaki linklerden yüklenir
- Farabi için yerel görsel kullanılır (src/farabi.png)
- Statik HTML, CSS ve JavaScript ile hazırlanmıştır
- Ekstra framework kullanılmaz

## Teknolojiler

- HTML
- CSS
- JavaScript

## Görsellerin Yüklenmesi

Görsel linkleri `src/links.json` dosyasında tutulur.  
Sayfa yüklendiğinde JavaScript bu dosyayı okur ve görselleri ilgili kartlara yerleştirir.  
Farabi için yerel dosya kullanılır.
